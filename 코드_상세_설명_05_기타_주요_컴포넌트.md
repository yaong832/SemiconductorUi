## 1. SimulationController

**파일 경로:** `SemiconductorUi/Controllers/SimulationController.cs`  
**역할:** 시뮬레이션 상태 및 타이머 관리

### 1.1 주요 책임

1. **시뮬레이션 타이머 관리**: 시뮬레이션 모드와 하드웨어 모드별 타이머 주기 설정
2. **경과 시간 추적**: 공정 경과 시간 관리
3. **상태 관리**: 실행, 일시정지, 중지 상태 관리

### 1.2 주요 메서드

```csharp
public void Start(bool isHardwareMode)
{
    // 하드웨어 모드: 150ms 주기
    // 시뮬레이션 모드: 1000ms 주기
    _simulationTimer.Interval = isHardwareMode 
        ? AppSettings.HardwareModeTickMilliseconds 
        : AppSettings.SimulationTickMilliseconds;
    _simulationTimer.Start();
}

public void Pause()
{
    _isPaused = true;
    _simulationTimer.Stop();
}

public void Resume(bool isHardwareMode)
{
    _isPaused = false;
    // 타이머 주기 재설정
    _simulationTimer.Start();
}

public void Stop()
{
    _isRunning = false;
    _isPaused = false;
    _simulationTimer.Stop();
}
```

### 1.3 IDisposable 구현

```csharp
public void Dispose()
{
    Dispose(true);
    GC.SuppressFinalize(this);
}

protected virtual void Dispose(bool disposing)
{
    if (!_disposed)
    {
        if (disposing)
        {
            _simulationTimer?.Stop();
            _simulationTimer?.Dispose();
        }
        _disposed = true;
    }
}
```

### 5.3 INotifyPropertyChanged 구현

```csharp
public event PropertyChangedEventHandler PropertyChanged;

protected virtual void OnPropertyChanged([CallerMemberName] string propertyName = null)
{
    PropertyChanged?.Invoke(this, new PropertyChangedEventArgs(propertyName));
}

protected bool SetProperty<T>(ref T field, T value, [CallerMemberName] string propertyName = null)
{
    if (EqualityComparer<T>.Default.Equals(field, value))
    {
        return false;
    }
    field = value;
    OnPropertyChanged(propertyName);
    return true;
}
```
